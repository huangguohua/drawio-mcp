# Draw.io Diagram Generation

When the user asks you to create a diagram, generate an **HTML artifact** that compresses the diagram and shows a button to open it in draw.io.

## HTML Artifact Code

Create an artifact with this code. Modify `diagramType` and `diagramCode` for your diagram:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <script src="https://cdnjs.cloudflare.com/ajax/libs/pako/2.1.0/pako.min.js"></script>
  <style>
    body { font-family: system-ui, sans-serif; padding: 40px; text-align: center; }
    .btn { display: inline-block; background: #0066cc; color: white; padding: 16px 32px;
           border-radius: 8px; text-decoration: none; font-size: 18px; font-weight: 500; }
    .btn:hover { background: #0052a3; }
    .info { margin-top: 20px; color: #666; font-size: 14px; }
    .error { color: #c00; }
  </style>
</head>
<body>
  <div id="result"></div>

  <script>
    // ========== DIAGRAM CONFIGURATION ==========
    const diagramType = "xml"; // Options: "xml", "mermaid", "csv"
    const diagramCode = `<mxGraphModel>
  <root>
    <mxCell id="0"/>
    <mxCell id="1" parent="0"/>
    <mxCell id="2" value="Hello" style="rounded=1;fillColor=#dae8fc;strokeColor=#6c8ebf;" vertex="1" parent="1">
      <mxGeometry x="100" y="100" width="120" height="60" as="geometry"/>
    </mxCell>
  </root>
</mxGraphModel>`;
    // ========== END CONFIGURATION ==========

    try {
      // Compress using pako
      const urlEncoded = encodeURIComponent(diagramCode);
      const compressed = pako.deflateRaw(urlEncoded);
      const b64 = btoa(String.fromCharCode.apply(null, compressed));

      // Build URL with lightbox mode
      const createObj = { type: diagramType, compressed: true, data: b64 };
      const jsonStr = JSON.stringify(createObj);
      const url = `https://app.diagrams.net/?lightbox=1&dark=auto&border=10&edit=_blank#create=${encodeURIComponent(jsonStr)}`;

      document.getElementById('result').innerHTML = `
        <a class="btn" href="${url}" target="_blank">Open in draw.io</a>
        <p class="info">Click to view diagram. Use the Edit button in draw.io to make changes.</p>
      `;
    } catch (e) {
      document.getElementById('result').innerHTML = `<p class="error">Error: ${e.message}</p>`;
    }
  </script>
</body>
</html>
```

## How It Works

1. Claude creates an HTML artifact with the diagram code
2. The artifact loads pako.js to compress the diagram
3. A button is shown that opens draw.io with the diagram in lightbox mode
4. In draw.io, hover to see the "Edit" button to open the full editor

## Diagram Examples

### Flowchart (Mermaid)

```javascript
const diagramType = "mermaid";
const diagramCode = `graph TD
    A[Start] --> B{Decision}
    B -->|Yes| C[Action 1]
    B -->|No| D[Action 2]
    C --> E[End]
    D --> E`;
```

### Sequence Diagram (Mermaid)

```javascript
const diagramType = "mermaid";
const diagramCode = `sequenceDiagram
    autonumber
    participant Client
    participant API
    participant Database

    Client->>API: POST /login
    API->>Database: Query user
    Database-->>API: User data

    alt Valid credentials
        API-->>Client: 200 OK + Token
    else Invalid credentials
        API-->>Client: 401 Unauthorized
    end`;
```

### ER Diagram (Mermaid)

```javascript
const diagramType = "mermaid";
const diagramCode = `erDiagram
    CUSTOMER ||--o{ ORDER : places
    ORDER ||--|{ LINE-ITEM : contains
    CUSTOMER {
        int id PK
        string name
        string email
    }
    ORDER {
        int id PK
        date created
        int customer_id FK
    }`;
```

### Custom Layout (XML)

```javascript
const diagramType = "xml";
const diagramCode = `<mxGraphModel>
  <root>

    </mxCell>
    <mxCell id="3" value="Process" style="rounded=1;fillColor=#dae8fc;strokeColor=#6c8ebf;" vertex="1" parent="1">
      <mxGeometry x="250" y="110" width="120" height="60" as="geometry"/>
    </mxCell>
    <mxCell id="4" value="End" style="ellipse;fillColor=#f8cecc;strokeColor=#b85450;" vertex="1" parent="1">
      <mxGeometry x="450" y="100" width="80" height="80" as="geometry"/>

    <mxCell id="5" style="endArrow=classic;" edge="1" parent="1" source="2" target="3">

```

### Org Chart (CSV)

```javascript
const diagramType = "csv";

```

## Quick Reference

| Diagram Type | diagramType | Best For |
|--------------|-------------|----------|
| Flowchart | `"mermaid"` | `graph TD` or `graph LR` |
| Sequence | `"mermaid"` | `sequenceDiagram` |
| ER Diagram | `"mermaid"` | `erDiagram` |
| Class Diagram | `"mermaid"` | `classDiagram` |
| Gantt Chart | `"mermaid"` | `gantt` |
| Custom Layout | `"xml"` | Precise positioning |
| Org Chart | `"csv"` | Hierarchical data |

## URL Parameters

The URL uses draw.io's lightbox mode:
- `lightbox=1` - Enables lightbox/preview mode
- `dark=auto` - Adapts to system dark/light preference
- `border=10` - Adds 10px border around diagram
- `edit=_blank` - Shows edit button that opens editor in new tab

## Notes

- Click the button to open draw.io with the diagram
- In draw.io lightbox, hover to see the "Edit" button
- Works with any diagram size - compression handles large XML
- Mermaid is recommended for most diagram types
